ORJ Job Board

Description

  • Project Link

ORJ is a job board where you can apply to jobs, post jobs, have resumes sent through ORJ to you through AWS SES, subscribe to new job postings through RSS feed. The project itself now discontinued was hosted on AWS Elastic Beanstalk.

Technology

  • Angular-JS
  • Node JS
  • AWS
  • Brain Tree API
