This is probably the biggest project that I've took part in creating. I was not only a developer on this project, but also a Project Manager, Business Analyst, Tester, Digital Marketer, Designer, and Salesperson. The website was at one time a upcoming hip hop forum that I later sold to a competitor in the same niche. It stood out from other forums mostly for the technology used and the fact we built this not using the typical forum software, but building from scratch. Buidling from scratch allowed us more freedom to customize and made for a better user experience.
